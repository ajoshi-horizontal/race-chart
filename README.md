## Getting Started

First, install packages:

```bash
npm i
```
then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

This app visualizes country population trends over time as an animated race chart, showing how rankings change year by year.
Users can play, pause, and step through the timeline while comparing the top 10 or top 15 countries by population.
